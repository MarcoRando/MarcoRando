# Hi, I'm Marco Rando 👋
Postdoctoral Researcher in Computer Science and Applied Mathematics   
INRIA & LJAD (CNRS) - Université Côte d’Azur

My research focuses on black-box and zeroth-order optimization, i.e., designing, implementing, and analyzing optimization algorithms in settings where first-order information (gradients) is not available. I study these algorithms under a wide range of settings, including deterministic and stochastic environments, non-smooth objectives, compositional functions, and bilevel optimization problems.

I also work on applying machine learning techniques to various domains, including reinforcement learning for navigation, machine learning for biology, high-energy physics, and robotics. Additionally, I investigate adversarial machine learning across different applications
